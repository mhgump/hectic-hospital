## Rules
- Every shipped non-code asset must be listed here (models, textures, sounds, fonts, UI art).
- Prefer assets with explicit internal redistribution rights.
- If attribution is required by the license, include the exact attribution text here and in any other required location.
- If an asset is “temporary placeholder”, mark it clearly and track the replacement plan.
